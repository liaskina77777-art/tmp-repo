# В данном ДЗ я выполняла задание с реализацией игры GGTP

Игра представляет собой угадывание чисел поврех протокола UDP с обработкой повторных передач, таймаутов. Также тут была реализована синхронная и асинхронная обработка - для работы нескольких игрков одновременно.
 
## Спецификация протокола GGTP

*HELO* - инцициализация игры
*WLCM a b* - приветствие с диапозоном a b
*GUES N* - предположение числа N
*MORE* - загаданное число больше
*LESS* - загаданное число меньше 
*BING* - победа, ключ
*FAIL* - проигрыш(больше нет попыток)

**./ggtp_client.py** - клиентская часть
**./ggtp_server.py** - синхронный сервер
**./ggtp_async_server.py** - асинхронный сервер

В целом, при игре попыток хватает для угадывания числа.  

Максимальное количество попыток = ceil(log2(размер диапозона))
Секретное число генерируется с использованием IP как сида
Победный ключ состоит из SHA256 хэш от IP + секретное число
